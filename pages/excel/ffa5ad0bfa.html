BETAINV

BETAINV 函数可得出给定累积 Beta 分布概率值的逆运算。所有参数都是数字值。

BETAINV(概率, alpha, beta, x 下限, x 上限)

概率： 与分布关联的概率。概率必须大于 0 且小于 1。

alpha： 分布的形状参数之一。alpha 必须大于 0。

beta： 分布的形状参数之一。beta 必须大于 0。

x 下限： 指定 x 值或概率的可选下限。 x 下限必须小于或等于概率。如果省略 x 下限，则使用 0。

x 上限： 指定 x 值或概率的可选上限。 x 上限必须大于或等于概率。如果省略 x 上限，则使用 1。

示例

=BETAINV(0.5, 1, 2, 0.3, 2) 得出 0.797918471982869。

=BETAINV(0.99, 1, 2, 0, 1) 得出 0.9。

=BETAINV(0.1, 2, 2, 0, 2) 得出 0.391600211318183。
