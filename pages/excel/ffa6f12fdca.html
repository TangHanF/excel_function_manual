SERIESSUM

SERIESSUM 函数用于计算和得出一个乘幂序列的和。这些系数用于通过步长值来递增的 x 值的连续乘幂。

SERIESSUM(x 值, 次方, 步长值, 系数)

x 值： 乘幂序列的 x 值输入。x 值是一个数字值。

次方： 表示为底数 x 值乘方的初始幂次的数字值。

步长值： 数字值，表示为序列中每项增加幂次的步进值。

系数： 系数将乘以 x 值的每个连续乘幂。系数的个数决定乘幂序列的项数。系数是一个数字值集合。

注释

  • SERIESSUM 函数支持任意的起始点和任意的指数间距。如果不要求以上任意一项，可考虑使用 POLYNOMIAL 函数。

  • 高阶项最后表示。系数 (ai) 用于 x 值的连续乘幂。在以下公式中，“x”代表 x 值，“n”代表次方，而“m”代表步长值。SERIESSUM 函数使用的公式是：a1 xn + a2 x(n+m) + a3 x(n+2m) + ...+ ak x(n + (k-1)m)，这里 k 表示系数的个数。

  • 指数必须是非负整数。

  • iBooks Author 中该函数不可用。

示例

=SERIESSUM(2, 0, 1, 9) 得出 9。

=SERIESSUM(2, 0, 1, 1) 得出 1。
另请参阅POLYNOMIAL
