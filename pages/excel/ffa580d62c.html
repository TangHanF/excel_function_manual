IFERROR

如果给定的值计算出错，IFERROR 函数将得出您指定的值；否则得出给定的值。

IFERROR(任何表达式, 出错返回值)

任何表达式： 要测试的表达式。任何表达式可以包含任意值。

出错返回值： 任何表达式计算出错时得出的值。出错返回值可以包含任何值。

注释

  • 使用 IFERROR 处理公式中的错误。例如，在处理的数据中，如果单元格 D1 的一个有效值为 0，公式 =B1/D1 将会出错（除数为零）。通过使用公式 =IFERROR(B1/D1, 0) 可防止出现此错误，它可以得出实际除数（如果 D1 不为 0）；否则得出 0。

示例

如果 B1 为数字值，D1 求得的值为 0，则：

=IFERROR(B1/D1, 0) 得出 0，因为以零作除数将导致出错。

=IF(ISERROR(B1/D1), 0, B1/D1) 等同于上一个 IFERROR 示例，但是需要使用 IF 和 ISERROR 函数。

如果单元格 B4 和 B5 都含有数字，=IF(IFERROR(OR(ISEVEN(B4+B5), ISODD(B4+B5)), FALSE), “All numbers”, “Not all numbers”) 返回文本“All numbers”；否则返回文本“Not all numbers”。这可以通过测试以查看这两个单元格之和是偶数还是奇数实现。如果单元格不是数字，EVEN 和 ODD 函数得出错误，IFERROR 函数得出 FALSE；否则得出 TRUE，因为 EVEN 或 ODD 为 TRUE。因此，如果 B4 或 B5 不是数字或布尔值，则 IF 语句返回为 FALSE 时返回的值表达式“Not all numbers”；否则返回为 TRUE 时返回的值表达式“All numbers”。
