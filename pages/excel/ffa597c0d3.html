OFFSET

OFFSET 函数返回与指定的基准单元格相隔指定行数和列数的单元格数组。

OFFSET(基准, 行偏移, 列偏移, 行数, 列数)

基准： 以其为基准测量偏移的单元格的单元格引用。

行偏移： 从基准单元格至目标单元格的行数。行偏移是一个数字值。0 表示目标单元格与基准单元格在同一行。负数表示目标单元格在基准单元格上面的行中。

列偏移： 从基准单元格到目标单元格的列数。列偏移是一个数字值。0 表示目标单元格与基准单元格在同一列。负数表示目标单元格在基准单元格左边的列中。

行数： 可选数字值，指定要得出的引用行数（从偏移位置开始计算）。

列数： 可选数字值，指定要得出的引用列数（从偏移位置开始计算）。

示例

假设您已在单元格 D7 中输入 1、在单元格 D8 中输入 2、在单元格 D9 中输入 3、在单元格 E7 中输入 4、在单元格 E8 中输入 5 且在单元格 E9 中输入 6。

在单元格 B6 中输入 =OFFSET(D7, 0, 0, 3, 1) 得出错误，因为得出的 3 行和 1 列（集合是 D7:D9）与 B6 没有单个的交汇单元格（一个都没有）。

在单元格 D4 中输入 =OFFSET(D7, 0, 0, 3, 1) 得出错误，因为得出的 3 行和 1 列（集合是 D7:D9）与 D4（它有三个）没有单个的交汇单元格。

在单元格 B8 中输入 =OFFSET(D7, 0, 0, 3, 1) 得出 2，因为得出的 3 行和 1 列（集合是 D7:D9）与 B8 具有单个的交汇单元格（即含有 2 的单元格 D8）。

在单元格 B7 中输入 =OFFSET(D7:D9, 0, 1, 3, 1) 得出 4，因为得出的 3 行和 1 列（集合是 E7:E9）与 B7 具有单个的交汇单元格（即含有 4 的单元格 E7）。

以下例子没有使用先前例子所述的假定的单元格内容。

=OFFSET(A1, 5, 5) 得出单元格 F6 中的值，因为该单元格正是在单元格 A1 的右边五列和下面五行。

=OFFSET(G13, 0, -1) 得出 G13 左边的单元格中的值，即 F13 中的值。

=SUM(OFFSET(A7, 2, 3, 5, 5)) 得出单元格 D9 至 H13 中值的总和，这些单元格是指以单元格 A7 右边两行和下面三列开始的五行和五列。

=SUM(OFFSET(INDIRECT(A1), 0, 0, A2, A3)) 得出在 A2 和 A3 中分别指定的行数和列数之和，其中从在 A1 中指定的基准单元格开始。
