BETADIST

BETADIST 函数可得出累积 Beta 分布概率值。所有参数都是数字值。

BETADIST(x 值, alpha, beta, x 下限, x 上限)

x 值： 要在 x 处为此函数求值的 x 值。x 值必须在范围 0 到 1 内。

alpha： 分布的形状参数之一。alpha 必须大于 0。

beta： 分布的形状参数之一。beta 必须大于 0。

x 下限： 指定 x 值或概率的可选下限。 x 下限必须小于或等于x 值。如果省略 x 下限，则使用 0。

x 上限： 指定 x 值或概率的可选上限。 x 上限必须大于或等于x 值。如果省略 x 上限，则使用 1。

示例

=BETADIST(0.5, 1, 2, 0.3, 2) 得出 0.221453287197232。

=BETADIST(1, 1, 2, 0, 1) 得出 1。

=BETADIST(0.1, 2, 2, 0, 2) 得出 0.00725。
